# perfect-agent

Starter scaffold for a deterministic, tool-using assistant.

## Structure

```text
perfect-agent/
├── agent/
│   ├── system_prompt.txt
│   ├── runner.py
│   └── tools/
│       ├── file.py
│       ├── http.py
│       └── shell.py
├── examples/
│   ├── analyze_repo.md
│   └── generate_docs.md
└── README.md
```

## Quick Start

```bash
cd perfect-agent/agent
python runner.py
```
